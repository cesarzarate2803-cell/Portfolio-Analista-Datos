**Proyecto 3: Sistema de Análisis de Datos - Cadena Veterinaria**
--

**Vista General del Proyecto**
Sistema completo de análisis de datos end-to-end para una cadena de clínicas veterinarias con presencia en 10 ciudades del Perú. Este proyecto demuestra competencias avanzadas en:

Diseño de Base de Datos - Arquitectura OLTP y Data Warehouse
SQL Avanzado - CTEs, Window Functions, optimización de queries
Python & ETL - Pipeline automatizado con 150K+ registros
Tableau Desktop - Dashboards ejecutivos profesionales
Modelado Dimensional - Esquema Estrella (Kimball Methodology)

--

**Objetivos del Proyecto**
Objetivo Principal
Diseñar e implementar una arquitectura completa de análisis de datos que permita a la gerencia tomar decisiones estratégicas basadas en información actualizada sobre operaciones, ventas y servicios veterinarios.
Objetivos Específicos

- Crear un Data Warehouse con modelo dimensional optimizado
- Generar 150,000+ registros realistas para análisis robusto
- Desarrollar pipeline ETL automatizado con Python
- Implementar dashboards ejecutivos en Tableau Desktop
- Documentar arquitectura y queries para replicabilidad
